# HTML/CSS/JS Projects

A collection of frontend projects built with vanilla HTML, CSS, and JavaScript.

---

## 1. Quiz Game
An interactive quiz app with multiple choice questions and instant feedback.

**Features:**
- 5 multiple choice questions
- Progress bar showing current question
- Green/red highlight on correct/wrong answers
- Score tracking
- Results screen with final score and restart option

**Tech:** HTML, CSS, JavaScript

---

## 2. Color Palette Generator
Generates random color palettes with one click.

**Features:**
- Generates 5 random colors at once
- Displays HEX codes for each color
- Copy color code with one click
- New palette on every click

**Tech:** HTML, CSS, JavaScript

---

## 3. Todo App
A clean task manager with filtering and completion tracking.

**Features:**
- Add and delete tasks
- Mark tasks as completed
- Filter by All / Active / Completed
- Shows current date
- Clear all completed tasks at once

**Tech:** HTML, CSS, JavaScript
